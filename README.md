# Capstone Project – Food Delivery Website

## Project Title & Theme
*Food Delivery Website* – A responsive website for online food ordering showcasing categories, restaurants, offers, testimonials, and mobile app download section.

## Short Description of Sections Implemented
1. *Categories Section* – Displaying different food types for user selection.  
2. *Restaurants Section* – List of available restaurants with images and ratings.  
3. *Offers Section* – Highlighting current discounts and deals.  
4. *Testimonials Section* – Customer reviews to increase trust.  
5. *Mobile App Download Section* – Promotes downloading the food delivery mobile app.

## Breakpoints & Main Layout Techniques
- *Breakpoints used:* 768px (tablet), 480px (mobile)  
- *Layout techniques:* CSS Flexbox, CSS Grid for responsive and structured layout
